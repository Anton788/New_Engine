// О том как собирать проект в Microsoft Visual Studio
1. Скачать архив Bh_Engine_VS
2. Установить Microsoft Visual Studio
3. Перекинуть папку Bh_Engine_WOC в удобное для вас место ( По умолчанию Visual Studio сохраняет проекты в C:\Users\User\source\repos)
4. Переместить папку VS_Libs в C:\
5. Переместить содержимое папки Graphics в C:\
6. Открыть в папке Bh_Engine_WOC Bh_engine_WOC.sln
7. Ткнуть правой кнопкой мыши в проект в обозревателе решений и открыть свойства
8. Открыть Каталоги VC++ и добавить во включаемые каталоги C:\VS_libs\include, а в каталоги библиотек C:\VS_libs\libs
9. Открыть Компоновщик -- Ввод --  Дополнительные зависимости и добавить следующие:
opengl32.lib
glfw3.lib
glew32s.lib
glut32.lib
SOIL.lib
10. Сохранить и скомпилировать проект
